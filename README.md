# VoiceFlow (声流) - 让你的声音，流淌成最优雅的文字 (v1.2)

VoiceFlow 是一款专为 macOS 设计的高效语音输入与智能润色工具。它不仅仅是一个语音转文字的工具，更是一个能理解你语义、实时纠错并优雅呈现的写作助手。

## ✨ 核心亮点

### 1. 智能差分注入 (Smart Differential Injection)
这是 VoiceFlow 与其他语音输入法最大的区别。在实时语音识别过程中，ASR 引擎往往会根据后续语境修正前面的内容。VoiceFlow 能够**实时计算字符差异**，自动执行退格键擦除并重新输入修正后的文字，实现真正的“流式输入，实时纠正”。

### 2. 极简主义 Aurora HUD
采用 macOS 最前沿的 `.ultraThinMaterial` 磨砂玻璃材质，搭配基于 Metal 渲染的 **Aurora 流体声纹动画**。200 个动态粒子环绕呼吸，让录音过程变成一种视觉享受。
- **焕彩主题**：提供紫罗兰、深海蓝、翡翠绿、极光等 **4 套高级配色方案**。
- **动态效果**：支持流畅、活力、宁静等 **3 种不同节奏的声纹动效**。
- **全局色彩同步**：界面细节（图标、滑块、菜单亮点）将随所选配色实时同步，沉浸感极佳。
- **自定义缩放**：支持 **0.5x 到 2.0x** 的整体比例缩放。

### 3. 多模态 ASR 引擎
- **原生模式**：追求极致响应与隐私，使用 Apple 系统级 ASR 引擎。
- **云端模式**：追求极致精准，完美支持 OpenAI Whisper 等主流云端识别接口。

### 4. 大模型智能润色
内置深度集成的 LLM 处理逻辑（支持 DeepSeek 等主流模型），提供多种模式：
- **基础精准**：去除语气词，修正同音字，保持原意。
- **正式润色**：将口语化的表达转化为规范的正式文档。
- **智能总结**：快速提取语音中的核心要点。

此外，HUD 菜单会通过**与主题色同步的微光圆点**实时提示当前激活的润色模式，操作直观。

## 📥 首次运行与授权指南 (新手必看)

由于本应用未在 App Store 上架，首次运行请务必遵循以下步骤：

1.  **解压并移动**：解压下载的 `.zip` 包，将 **VoiceFlow.app** 移动到 **「应用程序 (Applications)」** 文件夹中。
2.  **绕过安全检查**：**不要直接双击**。请「右键点击图标 -> 选择打开」，在弹出的对话框中再次点击「打开」。
3.  **授予核心权限**：
    -   **麦克风 & 语音识别**：在首次录音时点击“允许”。
    -   **辅助功能 (Accessibility)**：这是文字注入的核心。点击“打开系统设置”，在列表中勾选 VoiceFlow。如果已勾选但无效，请用底部的「减号 -」移除后重新添加。
4.  **配置 AI (可选)**：点击 HUD 的齿轮图标，在“模型设置”中配置您的 API Key。

## 🛠️ 如何使用

1.  **全局唤起**：按下 `Command + Shift + Space` 即可开启/停止录音。
2.  **实时预览**：在输入框内，你会看到文字正伴随着你的呼吸实时跃然屏上。
3.  **模式切换**：点击 HUD 上的 sparkles 图标，可快速切换润色模式（基础/正式/总结），选中的模式会带有高亮小点。
4.  **配置与自定义**：点击齿轮图标进入设置页。通过**标签页**在“模型设置”与“UI设置”间自由切换，配置 LLM 密钥或实时调节配色、动效及缩放比例。

## 🧩 兼容性与流式输出说明

VoiceFlow 具有两种输入模式，系统会根据目标应用自动切换，以确保输入的准确性：

### 1. 支持“流式实时输入” (Streaming)
这类应用遵循标准 macOS 辅助功能协议，能够完美呈现边说边打、由于 ASR 修正导致的自动实时回删纠错效果：
- **原生应用**：备忘录 (Notes)、文本编辑 (TextEdit)、邮件 (Mail) 等。
- **效率工具**：Antigravity。
- **浏览器**：Chrome、Safari 中的标准网页输入框。

### 2. 仅支持“全量最终输入” (Final Injection)
对于以下应用，VoiceFlow 会在录音结束后一次性注入润色后的文字。这是因为这些应用采用了自定义渲染引擎或特殊的 DOM 模拟机制，对大量连续的退格键（Backspace）响应不够及时，实时回删容易造成文字混乱：
- **终端工具**：**Warp**、iTerm2（高度自定义的 GPU 渲染引擎）。
- **Electron 应用**：**Obsidian**、VS Code（自定义文本缓冲区，不完全兼容标准的 AXUI 实时属性操作）。

## 🔨 开发者指南 (编译与打包)

项目基于 **Swift 6.0** 开发，并使用脚本一键打包。

### 环境要求
- macOS 14.0+ (Sonoma 及以上)
- Xcode 15.0+ 或 Swift 命令行工具

### 编译与运行
在终端执行以下脚本：
```bash
# 给予脚本权限
chmod +x package.sh

# 一键编译、生成图标并打包为 VoiceFlow.app
./package.sh

# 运行应用
open VoiceFlow.app
```
